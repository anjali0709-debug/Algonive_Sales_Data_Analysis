# Sales Data Analysis and Forecasting

## 📊 Project Overview
This project analyzes retail sales data and predicts future sales trends using time series forecasting.

## 🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Sales Forecasting using ARIMA
- Data Visualization

## 📁 Dataset
Retail Sales Dataset (CSV)

## 🛠 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels
- Plotly

## 📈 Output
- Sales trends visualization
- Category-wise analysis
- Forecast of next 30 days
